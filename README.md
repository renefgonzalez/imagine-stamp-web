# Imagine & Stamp - Boutique de Diseño y Personalizados

Bienvenido al repositorio oficial de **Imagine & Stamp**, una boutique digital especializada en la creación de momentos mágicos a través del diseño y la personalización de alta calidad.

## 🎨 Sobre el Proyecto

Esta aplicación es una plataforma de catálogo interactivo diseñada para ofrecer una experiencia de compra tipo "boutique". Permite a los usuarios explorar diversas categorías de productos personalizados, desde invitaciones digitales hasta kits corporativos, con una interfaz fluida, moderna y profesional.

### Características Principales:

- **Catálogo Dinámico**: Navegación intuitiva por categorías (Invitaciones, Fiestas, Eventos, Corporativo).
- **Mega Menú Boutique**: Organización elegante de subcategorías para una búsqueda precisa.
- **Panel de Administración**: Gestión completa de productos y categorías con persistencia local.
- **Integración con WhatsApp**: Botón de cotización directa y finalización de pedidos vía chat.
- **Diseño Responsivo**: Optimizado para dispositivos móviles y escritorio utilizando Tailwind CSS.
- **Animaciones Fluidas**: Experiencia de usuario mejorada con Framer Motion.

## 🛠️ Tecnologías Utilizadas

- **React 19**: Biblioteca principal para la interfaz de usuario.
- **Vite**: Herramienta de construcción ultra rápida.
- **Tailwind CSS 4**: Framework de utilidades CSS para un diseño moderno y consistente.
- **Lucide React**: Set de iconos elegantes y minimalistas.
- **Framer Motion**: Para micro-interacciones y transiciones suaves.
- **TypeScript**: Para un desarrollo robusto y tipado.

## 🚀 Despliegue

El proyecto está configurado para ser desplegado en plataformas de hosting estático (como GitHub Pages, Vercel o Netlify).

### Configuración de Dominio Personalizado:

El archivo `CNAME` en la carpeta `public` ya está configurado para el dominio: `imagineandstamp.com`.

## 📦 Instalación y Desarrollo

Si deseas ejecutar este proyecto localmente:

1. Clona el repositorio.
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
4. Para construir la versión de producción:
   ```bash
   npm run build
   ```
